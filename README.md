# Oi, eu sou a Ingrid Vanzeli

Analista de QA, atuando com testes manuais, automação de testes Web e testes de API. Entrei como analista de sistemas, atuando na prática com suporte técnico, e migrei para qualidade em 2024 — o que me deu uma base sólida em investigação de causa raiz, análise de logs e SQL, algo que uso até hoje no dia a dia de testes. Além, é claro, de desenvolver um pensamento crítico e entender como um pequeno bug pode impactar no dia a dia dos usuários.

Aqui no GitHub compartilho projetos de estudo e prática em automação, aplicando o que aprendo no meu trabalho e além dele.

## Sobre mim

- Foco em teste de software e garantia da qualidade, com atuação profissional em testes manuais e automatizados
- Automação de testes Web com Selenium WebDriver, Cucumber/Gherkin e TestNG, com gestão de evidências via Allure Reports
- Testes de API com Postman no dia a dia profissional, e estudos com Cypress e Rest Assured
- Experiência prática com cenários de login, cadastro, autenticação e validação de endpoints
- Gerenciamento de massa de testes via SQL/MariaDB e acompanhamento de pipelines de integração contínua com Jenkins
- Estudos em automação Web, API e Performance (pretendo estudar automação mobile futuramente)
- Interesse em boas práticas de teste, critérios de aceite, levantamento de requisitos e documentação
