# Oi, eu sou a Ingrid Vanzeli

Sou estudante e praticante de Teste de Software, com foco em qualidade, automacao de testes e validacao de aplicacoes web e APIs. No meu GitHub, compartilho projetos de estudo e pratica usando ferramentas como Selenium, Cypress, Playwright, Rest Assured, Postman e k6.

Gosto de transformar cenarios de teste em suites organizadas, com uma visao bem pratica: validar fluxos importantes, encontrar falhas cedo e ajudar a entregar software com mais confianca.

## Sobre mim

- Foco em Teste de Software e Garantia da Qualidade
- Estudos e projetos em automacao web, API e performance
- Experiencia pratica com cenarios de login, cadastro, autenticacao e validacao de endpoints
- Interesse em boas praticas de testes, criterios de aceite, levantamento de requisitos e documentacao
